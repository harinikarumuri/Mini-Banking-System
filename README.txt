# SwiftBank ATM System

## Description
SwiftBank is a simple console-based ATM simulation project built using Python. It allows a user to:
- Login with credentials
- Check balance
- Deposit money
- Withdraw money

## How to Run
1. Make sure you have Python installed.
2. Run the script using:
   ```
   python swiftbank_atm.py
   ```

## Note
- All user data is hardcoded for simplicity.
- This project is meant for learning and practice purposes only.
